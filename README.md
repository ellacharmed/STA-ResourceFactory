# STA-ResourceFactory
Creates a resource, using the Fish Factory mesh from the modtool sample. Clones 10 units of the following resources every 5(?) game-minutes (aka 300 of Production Time): 
- Antibiotics
- Cloth
- Component
- Concrete
- Debris
- Electronics
- Fiber
- Firewood
- Iodine
- Metal
- Parts
- Plank
- Plastic
- Tools
- Trash
- Wood

Located in Build menu under Resources tab. Does not need any input for cloning production process.

##  Requirements
* Build a General storage first
* Construction Cost: Metal 10 units
* Repair cost: Metal 1 unit


## Expected outcome
* Output: 10 units of the listed resources above
